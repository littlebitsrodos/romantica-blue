2. Quelles données nous collectons

a) Lorsque vous remplissez le formulaire de contact

  • Nom
  • E-mail
  • Dates d'intérêt (si sélectionnées dans le calendrier)
  • Le message que vous envoyez

b) Lorsque vous naviguez sur le site (uniquement avec votre consentement)

  • Données d'usage anonymes via Google Analytics 4 (pages vues, durée de session, pays/ville au niveau de la ville, type d'appareil)
  • Adresse IP — pseudonymisée par GA4

Nous ne collectons pas de données sensibles, de données de paiement (gérées directement par Stripe / Viva), ni de données publicitaires. Nous n'établissons aucun profil.

3. Pourquoi nous les collectons — base légale

  • Formulaire de contact : pour répondre à votre demande de réservation. Base légale : mesures précontractuelles à votre demande (article 6.1.b RGPD).
  • Analytique : pour comprendre comment le site est utilisé et l'améliorer. Base légale : consentement via le bandeau cookies (article 6.1.a RGPD).

4. Cookies

Nous utilisons uniquement des cookies analytiques — pas de publicité, pas de suivi inter-sites. Les cookies analytiques ne sont pas chargés tant que vous n'avez pas donné votre consentement explicite via le bandeau (Google Consent Mode v2 — refusé par défaut).

Cookie Finalité Durée

Vous pouvez modifier ou retirer votre consentement à tout moment en supprimant les cookies depuis les paramètres de votre navigateur.

5. Tiers (sous-traitants)

Nous partageons le minimum de données nécessaires avec les prestataires suivants :

  • Web3Forms (web3forms.com) — transmet les messages du formulaire à notre e-mail.
  • Google Ireland Ltd. — Google Analytics 4 (données pseudonymisées, uniquement après consentement).
  • GitHub Inc. / Microsoft Corp. — hébergement du site (GitHub Pages).
  • Stripe / Viva Payments — traitement des paiements si vous finalisez une réservation. Les données de carte ne transitent jamais par nous.

Lorsque vous cliquez sur des liens externes (Airbnb, Booking.com), leurs propres politiques de confidentialité s'appliquent — nous ne leur partageons pas vos données.

6. Transferts internationaux

Google, Web3Forms, GitHub et Stripe peuvent traiter des données sur des serveurs hors Espace Économique Européen (principalement aux États-Unis). Dans ce cas, le transfert est couvert par les Clauses Contractuelles Types approuvées par la Commission européenne et/ou des certifications telles que l'EU-US Data Privacy Framework.

7. Durée de conservation

  • Messages du formulaire : jusqu'à 24 mois après la dernière interaction, ou jusqu'à demande de suppression.
  • Données de réservation / paiement : selon la législation fiscale grecque (10 ans pour les factures).
  • Analytique : 14 mois (limite GA4).

  • À la limitation du traitement
  • À la portabilité des données
  • De vous opposer au traitement
  • De retirer votre consentement à tout moment
